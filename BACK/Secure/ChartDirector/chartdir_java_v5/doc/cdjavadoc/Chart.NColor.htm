ChartDirector Ver 5.0 (Java Edition)

Chart.NColor

Usage

public static java.awt.Color NColor(int color)

Description

Converts a ChartDirector color to a java.awt.Color object.

This method is the reverse of Chart.CColor.

Arguments

ArgumentDefaultDescription
color(Mandatory)A ChartDirector color.

Return Value

An java.awt.Color object.
