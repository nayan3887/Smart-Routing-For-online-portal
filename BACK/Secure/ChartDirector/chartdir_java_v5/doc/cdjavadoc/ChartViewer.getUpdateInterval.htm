ChartViewer.getUpdateInterval

Usage

public int getUpdateInterval()

Description

Gets the minimum gap interval between two ViewPortChanged events in milliseconds.

Arguments

None

Return Value

The minimum gap interval between two ViewPortChanged events in milliseconds. See ChartViewer.setUpdateInterval for the meaning of this value.
