Chart.MouseUsageScrollOnDrag

Declaration

public const int MouseUsageScrollOnDrag

Description

A constant equals to 1 to represent using the mouse for drag scrolling.

This constant is used in ChartViewer.setMouseUsage and WebChartViewer.setMouseUsage .
