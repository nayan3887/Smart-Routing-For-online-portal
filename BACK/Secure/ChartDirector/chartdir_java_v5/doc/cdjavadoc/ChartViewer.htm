ChartDirector Ver 5.0 (Java Edition)

ChartViewer

ChartViewer is a Java Bean and a SWING component for viewing ChartDirector charts and for providing tool tips and hot spots support. It is primarily used in standalone Java applications or in Applets. It is not needed for server side charting applications (Java servlets or JSP applications).

ChartViewer is a subclass of the SWING JLabel component.
