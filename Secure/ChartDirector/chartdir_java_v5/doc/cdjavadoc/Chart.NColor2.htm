ChartDirector Ver 5.0 (Java Edition)

Chart.NColor2

Usage

public static java.awt.Color[] NColor2(int[] colors)

Description

Converts an array of ChartDirector colors to an array of java.awt.Color objects.

This method is the reverse of Chart.CColor2.

Arguments

ArgumentDefaultDescription
colors(Mandatory)An array of ChartDirector colors.

Return Value

An array of java.awt.Color objects.
