ChartViewer.getHotSpotCursor

Usage

public void getHotSpotCursor(Cursor cursor)

Description

Gets the cursor that is displayed when the mouse cursor is over a normal hot spot.

Arguments

None

Return Value

The java.awt.Cursor object to display when the mouse is over a normal hot spot.
