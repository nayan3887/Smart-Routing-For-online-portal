HotSpotListener.hotSpotReleased

Usage

public void hotSpotReleased(HotSpotEvent e)

Description

The system will call this method when the mouse is over a hot spot region and a mouse button is released.

Arguments

ArgumentDefaultDescription
e(Mandatory)A HotSpotEvent object containing data of the hot spot event.

Return Value

None
