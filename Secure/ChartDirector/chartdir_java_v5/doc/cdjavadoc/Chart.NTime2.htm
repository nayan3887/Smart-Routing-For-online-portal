ChartDirector Ver 5.0 (Java Edition)

Chart.NTime2

Usage

public static java.util.Date[] NTime2(double[] times)

Description

Converts an array of dates/times in ChartDirector's date/time format to java.util.Date objects.

This method is the reverse of Chart.CTime2.

Arguments

ArgumentDefaultDescription
times(Mandatory)An array of numbers representing dates/times using ChartDirector's date/time format.

Return Value

An array of java.util.Date objects.
