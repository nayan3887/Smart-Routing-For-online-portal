ChartDirector Ver 5.0 (Java Edition)

Chart.NTime

Usage

public static java.util.Date NTime(double time)

Description

Converts a date/time in ChartDirector's date/time format to a java.util.Date object.

This method is the reverse of Chart.CTime.

Arguments

ArgumentDefaultDescription
time(Mandatory)A number representing date/time using ChartDirector's date/time format.

Return Value

A java.util.Date object.
