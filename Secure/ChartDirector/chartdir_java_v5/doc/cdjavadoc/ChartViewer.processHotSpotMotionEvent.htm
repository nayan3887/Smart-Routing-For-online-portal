ChartViewer.processHotSpotMotionEvent

Usage

protected void processHotSpotMotionEvent(HotSpotEvent e)

Description

Process hot spot mouse motion events.

The system will call this method whenever a hot spot mouse motion event occur. If you override this method, remember to invoke the superclass's processHotSpotMotionEvent method for proper functioning of hot spot user interactions.

Arguments

ArgumentDefaultDescription
e(Mandatory)The HotSpotEvent object containing data for the hot spot mouse motion event.

Return Value

None
